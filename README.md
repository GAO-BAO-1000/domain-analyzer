# Domain Analyzer

Консольное приложение на Python для получения информации о домене через WHOIS.

## Возможности

- получение владельца домена;
- получение регистратора;
- дата регистрации домена;
- дата окончания регистрации.

## Используемые технологии

- Python 3
- python-whois

## Запуск

```bash
python3 main.py
```

## Структура проекта

- main.py
- api_client.py
- data_processor.py
- report.py
